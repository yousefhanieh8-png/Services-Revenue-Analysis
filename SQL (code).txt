# Services Revenue Analysis (SQL)

## Overview
This notebook contains SQL queries used to analyze services revenue data.

---
-- Total Revenue
SELECT SUM(total_revenue) AS [Total Revenue]
FROM Services_data;


-- Total Hours
SELECT SUM(hours) AS [Total Hours]
FROM Services_data;


-- Revenue by Department
SELECT department, SUM(total_revenue) AS [Total Revenue]
FROM Services_data
GROUP BY department;


-- Revenue by Client
SELECT client_name, SUM(total_revenue) AS [Total Revenue]
FROM Services_data
GROUP BY client_name;


-- Revenue by Region
SELECT region, SUM(total_revenue) AS [Total Revenue]
FROM Services_data s
JOIN Branch_data b
ON s.Branch_ID = b.Branch_ID
GROUP BY region;
